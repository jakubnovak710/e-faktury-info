Peppol ID je unikátny identifikátor, ktorý jednoznačne identifikuje každého účastníka v sieti Peppol. Funguje ako "adresa" firmy v digitálnom svete e-fakturácie — bez Peppol ID nie je možné odosielať ani prijímať e-faktúry cez Peppol.

## Formát Peppol ID pre Slovensko

Pre slovenské firmy sa Peppol ID skladá z prefixu schémy a samotného identifikátora. Formát je `0245:XXXXXXXXXX`, kde `0245` je kód identifikačnej schémy pre slovenské DIČ (daň z pridanej hodnoty identifikačné číslo) a `XXXXXXXXXX` je 10-miestne DIČ firmy. Napríklad firma s DIČ 2021234567 má Peppol ID `0245:2021234567`.

## Registrácia Peppol ID

Peppol ID si firma nemôže zaregistrovať sama — registráciu vykonáva poskytovateľ Access Pointu (digitálny poštár). Pri registrácii sa údaje o firme zapíšu do registra SMP (Service Metadata Publisher), kde sú viditeľné pre všetkých účastníkov siete. Register obsahuje informácie o tom, aké typy dokumentov firma prijíma a cez ktorý Access Point.

## Zmena poskytovateľa

Ak firma chce zmeniť digitálneho poštára, jej Peppol ID zostáva rovnaké — zmení sa len priradenie k novému Access Pointu v registri SMP. Tento proces sa nazýva migrácia a mala by byť bezproblémová, ak si ju koordinujete s oboma poskytovateľmi.

## Overenie existencie Peppol ID

Pred odoslaním e-faktúry môžete overiť, či má prijímateľ platné Peppol ID. Existujú verejné Peppol Directory a lookup nástroje, kde zadávate identifikátor a zistíte, či je firma zaregistrovaná. To je užitočné najmä v prechodnom období, keď ešte nie všetky firmy budú mať Peppol ID. Pre slovenský trh je od roku 2027 Peppol ID povinné pre všetkých platiteľov DPH.
