Access Point (prístupový bod) je technický koncový bod, ktorý zabezpečuje pripojenie firmy do siete Peppol. Je to server prevádzkovaný certifikovaným poskytovateľom (digitálnym poštárom), ktorý odosiela a prijíma elektronické dokumenty v mene svojich klientov.

## Ako Access Point funguje

Keď firma potrebuje odoslať e-faktúru, jej účtovný systém odovzdá dokument Access Pointu odosielateľa. Ten najprv overí formát a validitu faktúry, potom vyhľadá v registri SMP (Service Metadata Publisher) Access Point prijímateľa a doručí mu dokument. Komunikácia medzi Access Pointmi prebieha cez zabezpečený protokol AS4 so šifrovaním a digitálnym podpisom.

## Certifikácia a bezpečnosť

Každý Access Point musí prejsť certifikačným procesom organizácie OpenPeppol. Poskytovateľ musí splniť prísne technické a bezpečnostné požiadavky, podpísať zmluvu s OpenPeppol a pravidelne prechádzať auditmi. Certifikát sa obnovuje každoročne. Na Slovensku je navyše potrebná akreditácia Finančnou správou SR.

## Access Point vs. digitálny poštár

Pojmy Access Point a digitálny poštár spolu úzko súvisia, ale nie sú identické. Access Point je technický termín pre serverovú infraštruktúru, zatiaľ čo digitálny poštár je obchodné označenie pre poskytovateľa, ktorý tento Access Point prevádzkuje a ponúka klientom služby doručovania e-faktúr.

## Význam pre podnikateľov

Podnikatelia si nemusia prevádzkovať vlastný Access Point — to by bolo finančne a technicky náročné. Namiesto toho si zvolia poskytovateľa (digitálneho poštára), ktorý prístupový bod prevádzkuje. Poskytovateľ zvyčajne ponúka API alebo webové rozhranie, cez ktoré môžete e-faktúry odosielať a prijímať. Niektorí poskytovatelia ponúkajú aj priamu integráciu s populárnymi účtovnými softvérmi na Slovensku.
